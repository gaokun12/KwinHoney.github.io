A review for iPad Folio Keyboard 12.9

June 17, 2021

This is my first time to use this keyboard. From my first impression and some limited experience about Apple keyboard Folio series. I think this keyboard is not suitable for a researcher student or programmer. But, after buying a Folio keyboard from a second-hand platform, I totally change my attitude about this kind of keyboard. I really like this keyboard!

  • I can use it in a lot of position just like I am using a laptop. I use it when I lying on the sofa, on my bed and on the regular desk.
  • The typing experience is good. Although it is different with using the keyboard of a Mac or Apple magical keyboard, but I really enjoy this kind of experience when I get to used it.
  • I can use this keyboard when I want note on e-papers, write easy documents in notions, and writing some easy code on Leecode.
