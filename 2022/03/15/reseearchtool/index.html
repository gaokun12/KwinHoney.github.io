Research Tools

March 15, 2022

In this article, I will list some useful command which compacts the codes in deep learning projects. The commands includes PyTorch, Numpy, and TensorFloe, etc.

  1. numpy.in1d (ar1, ar2, assume_unique=False, invert=False)[source]

    Test whether each element of a 1-D array is also present in a second array.Returns a boolean array the same length as ar1 that is True where an element of ar1 is in ar2 and False otherwise.We recommend using isin instead of in1d for new code.

    Tips:

    This commend can be used when produce false examples in relational graph based link prediction tasks.
